## 🚀 Featured Projects

### 🔍 [AI Automotive Log Analyzer](https://github.com/chiranjeevisafari/AI_Log_Anlayzer_Tool)
AI-powered full-stack application that analyzes automotive validation log files (500K+ lines) and produces structured, actionable issue reports.

**Key Features:**
- 🧠 AI analysis via GitHub Copilot CLI (Claude/GPT models)
- 🎫 Direct JIRA integration — one-click ticket creation with team bug template
- 🔄 Live JIRA bug sync — fetches 2000+ tickets for similarity matching
- 📊 Interactive web dashboard with severity breakdown and health scores
- ⚡ Smart caching — same file returns instant results (shared between web & terminal)
- 📋 Supports DLT, CAN/ASC, Android logcat, Jenkins console, and generic logs
- 📄 Export to PDF / Excel / CSV

**Tech:** Python, FastAPI, WebSocket, JavaScript, SQLite, TF-IDF/NLP, GitHub Copilot CLI

### 🤖 Selenium Automation Framework
Robust Selenium-TestNG-Cucumber framework built using Page Object Model and reusable utilities.

### 📱 Android UI Automation Framework
